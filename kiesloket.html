Schoolproject Semester 1, Jaar 2

Het Kiesloket

Een webapplicatie waarin gebruikers verkiezingsuitslagen kunnen bekijken en vergelijken per kieskring. Gebruikers kunnen zich registreren, inloggen en meningen delen via posts. Gebouwd met Vue en TypeScript op de frontend en Java met Spring Boot op de backend.

Java Spring Boot Vue TypeScript MySQL XML Parser REST API

Het project

Waar gaat het over?

Doel

Gebruikers kunnen verkiezingsuitslagen bekijken van 2021 en 2023, filteren per kieskring en de resultaten vergelijken. Ook kunnen ze posts maken om meningen te delen.

Technologie

Frontend in Vue met TypeScript. Backend in Java met Spring Boot en REST API's. Data opgeslagen in MySQL. Verkiezingsdata ingeladen via een XML parser die de officiële bestanden van de Tweede Kamer verwerkte.

Samenwerking

Team van 3 personen. Taken verdeeld via een issue board op GitLab met de MoSCoW methode. Elke sprint hadden we een review en een retrospective om te reflecteren op ons werk.

Mijn bijdrage

Wat heb ik gedaan?

01

Profielpagina

Voor elke pagina die ik heb gemaakt moest ik eerst een figma design maken. Ik heb de profielpagina gebouwd waar een ingelogde gebruiker zijn eigen gegevens kan bekijken zoals gebruikersnaam, e-mail en geboortedatum. Ook heb ik de bewerkfunctie gebouwd zodat een gebruiker zijn gegevens kan aanpassen.

02

Verkiezingsuitslagen pagina

Ik heb de pagina gebouwd waar gebruikers de uitslagen van 2021 en 2023 kunnen bekijken en vergelijken. Ook heb ik de kieskring naast elke partij zichtbaar gemaakt zodat gebruikers snel kunnen zien welke regio bij welke uitslag hoort.

03

Homepage

Ik heb de homepage gebouwd met een introductie van de website, een overzicht van de top 3 partijen per kieskring en navigatie naar de andere onderdelen zoals het forum en de verkiezingskaart.

04

XML Parser

Meegewerkt aan de XML parser die de officiële verkiezingsresultaten van de Tweede Kamer inlaadt en naar de MySQL database stuurt. Hierdoor stonden alle uitslagen automatisch klaar in de applicatie zonder dat je ze handmatig hoefde in te voeren.

05

Onderzoek ArrayList vs LinkedList

Ik heb een onderzoeksrapport geschreven over de prestatieverschillen tussen ArrayList en LinkedList in Java. Ik heb literatuuronderzoek gedaan en zelf code tests geschreven om de twee te vergelijken. De conclusie was dat ArrayList in de meeste gevallen sneller is door betere geheugenlocatie.
